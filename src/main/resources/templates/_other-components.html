Légende
V-E-H = Victoires de l'entraineur sur l'hippodrome (en pourcentage)
V-J-H = Victoires du jockey sur l'hippodrome (en pourcentage)
V-C-H = Victoires du cheval sur l'hippodrome (en pourcentage)
P-C-H = Places du cheval sur l'hippodrome (en pourcentage)
V-CO = Victoires de l'association jockey/cheval (en pourcentage)
P-CO = Places de l'association jockey/cheval (en pourcentage)
V-CO-H = Victoires de l'association jockey/cheval sur l'hippodrome (en pourcentage)
P-CO-H = Places de l'association jockey/cheval sur l'hippodrome (en pourcentage)
V-CH-PSF = Victoires du cheval sur PSF (en pourcentage)
P-CH-PSF = Places du cheval sur PSF (en pourcentage)
V-CH-PSF-H = Victoires du cheval sur PSF sur l'hippodrome (en pourcentage)
P-CH-PSF-H = Places du cheval sur PSF sur l'hippodrome (en pourcentage)
Étoiles
: Cheval figurant dans les 5 premiers, dans au moins 6 rubriques (7 dans les courses sur PSF)
: Cheval figurant dans les 3 premiers des rubriques "V-E-H" (Entraineurs)) et "V-J-H" (Jockeys)
Pastilles
: Cheval ne figurant pas dans la rubrique "V-E-H" (Entraineurs)
: Cheval ne figurant pas dans la rubrique "J-E-H" (Jockeys)
: Cheval n'ayant aucune expérience avec le jockey du jour
: Cheval n'ayant aucune expérience sur PSF
